# AURORA LOGIN PORTAL
## DESCRIPTION
This is a modern and visually stunning login page built using HTML AND CSS,featuring a beautiful glassmorphism design,animated gradinet background and smooth UI effects.

### FEATURES
- Glassmorphism login card
- Vibrant gradient background
- Responsive and centered layout
- Smooth hover animations
- Clean and structured form design

## TECHNOLOGIES USED
   HTML
   
   CSS

### HOW IT WORKS
1. The HTML provides the structure of the login form
2. The CSS handles:
    - Glass effect
    - Gradient background
    - Layout using Flexbox
    - Animations and Hover effects
